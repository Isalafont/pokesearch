# PokeSearch

Welcome to this Pokesearch application.
Enter the name of your pokemon and his card will be display.

This app is using the [PokeApi](https://pokeapi.co/)

### Stack :

- React App
- Tailwind
- Vercel

## Getting Started

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
